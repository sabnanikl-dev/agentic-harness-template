# Design Documentation

Place brand assets, inspiration screenshots, and visual references here.

## What Goes Here

- Brand guidelines (colors, typography, spacing tokens)
- Inspiration screenshots from reference sites
- Component mockups or wireframes
- Design system documentation

## Naming Convention

- `brand-guidelines.md` — official brand rules
- `inspiration-<site-name>.png` — screenshots of reference sites
- `mockup-<feature>.png` — wireframes or mockups
