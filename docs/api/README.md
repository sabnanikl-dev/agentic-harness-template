# API Documentation

Place API schemas, endpoint definitions, and integration docs here.

## What Goes Here

- OpenAPI / Swagger specs
- GraphQL schemas
- Endpoint documentation (request/response examples)
- Authentication flows
- Third-party API integration notes

## Naming Convention

- `openapi.yaml` or `swagger.yaml` — API spec
- `endpoints.md` — human-readable endpoint docs
- `integration-<service>.md` — third-party integration guides
